-------------------------------------------------------------
UPDATING
-------------------------------------------------------------
To update NodeCanvas you should always:

- Backup your project.
- Remove previous NodeCanvas folder.
- Reimport NodeCanvas package anew.

-------------------------------------------------------------
USING WITH FLOWCANVAS
-------------------------------------------------------------
If you have both NodeCanvas and FlowCanvas in your project, please do the following:
- Locate and delete the sub-folder named "Framework" either in 'NodeCanvas', or in 'FlowCanvas' main folder.
